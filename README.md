# Market7 - Intelligent Crypto Trading Pipeline

Market7 is an advanced trading system that combines fork filtering, smart DCA (Dollar-Cost Averaging), machine learning predictions, and recovery management to optimize trade entries and exits.

## Core Components
- **Fork Filtering** with ML boost and TV kicker logic
- **Smart DCA** rescue engine with health and recovery odds
- **BTC Market Filtering** for trend safety
- **Live Monitoring** and Auto-Logging
- **Dashboard Ready** with backend API routes and frontend UI

## Quick Start
1. Clone repository
2. Setup virtual environment
3. Install requirements: `pip install -r requirements.txt`
4. Configure YAML settings under `/config`
5. Launch trading scripts or dashboard backend

## Requirements
- Python 3.11+
- FastAPI
- ReactJS (Frontend)
- Redis
- 3Commas API Access
